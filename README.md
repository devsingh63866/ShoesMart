# 👟 ShoesMart - Modern Shoe Website

A modern and responsive e-commerce frontend website for showcasing shoes.  
Built using HTML, CSS, and Bootstrap with a clean black & white theme.

---

## 🚀 Live Demo
🔗 https://your-link.onrender.com

---

## 📌 Features

- 🖤 Modern Black & White UI  
- 📱 Fully Responsive Design  
- 🧭 Navbar with hover effects  
- 🛍️ Product Cards Section  
- 🖼️ Image Collage Layout  
- ℹ️ About Section  
- 🔗 Important Links (Footer)  
- ✨ Smooth animations  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Bootstrap 5
